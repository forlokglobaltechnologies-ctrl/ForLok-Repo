Week 1 Python Foundations Days 1–7
Day 1
Club Kick-off & Python Intro
Club Event
  • Official club launch ceremony — welcome all members
  • What is AI? What is ML? 10-minute visual intro
  • Install Python, VS Code, Anaconda
  • First Python program: Hello World + variables
  • Share club WhatsApp/Discord group
Day 2
Data Types & Variables
Learn
  • int, float, string, bool — types and casting
  • Variable naming conventions
  • Input/output functions: input(), print()
  • Mini exercise: simple calculator
  • 📣 Post Day 2 code snippet in group chat
Day 3
Loops & Control Flow
Learn
  • if / elif / else conditions
  • for loops and while loops
  • Nested loops — pattern printing
  • Logic problems: palindrome, even/odd, Armstrong
  • Fibonacci series implementation
Day 4
Functions — Part 1
Learn
  • Defining functions, return statements
  • Arguments vs parameters
  • Default arguments, keyword args
  • Lambda functions introduction
  • Exercise: refactor Day 3 solutions using functions
Day 5
Recursion
Learn
  • What is recursion? Base condition writing
  • Recursive logic vs iterative logic
  • Implement: factorial, Fibonacci using recursion
  • Visualise call stack with print statements
Day 6
Lists, Tuples & Strings
Learn
  • Lists — indexing, slicing, methods
  • Tuples — immutability and use cases
  • String methods and f-strings
  • List comprehensions introduction
Day 7
Week 1 Mini Challenge + Demo
Event
  • 🏆 30-min timed coding challenge — 3 problems
  • Each member presents 1 solution to the group
  • Peer review — what could be improved?
  • Senior mentor Q&A session (30 min)
  • Celebrate top 3 solutions publicly
Week 2 OOP & Problem Solving Days 8–14
Day 8
Dictionaries & Sets
Learn
  • Dictionaries — key-value pairs, methods
  • Sets — uniqueness, set operations
  • When to use which data structure
  • Exercise: word frequency counter
Day 9
OOP — Classes & Objects
Learn
  • What is a class? Blueprint analogy
  • Creating objects, __init__ constructor
  • Instance variables vs class variables
  • Build: Student class with name, grade, GPA
Day 10
Inheritance & Abstraction
Learn
  • Single and multilevel inheritance
  • super() keyword and method overriding
  • Abstract classes using ABC module
  • Exercise: Animal → Dog/Cat hierarchy
Day 11
Encapsulation & Polymorphism
Learn
  • Private and protected attributes
  • Getters and setters (properties)
  • Method overloading and overriding
  • Build: Bank Account class with encapsulation
Day 12
File Handling & Error Handling
Build
  • Reading and writing .txt and .csv files
  • try / except / finally blocks
  • Custom exceptions
  • Exercise: Save student records to file
  • 🗣 Guest talk: "Why Python in AI/ML?" (20 min)
Day 13
Problem Solving Session
Build
  • Apply OOP to 3 real-world problems
  • Think in classes: Library system, ATM, Inventory
  • Code review pairs — 2 members swap code
  • Introduce GitHub basics: commit + push
Day 14
Week 2 Review + Show & Tell
Event
  • 🎤 Show & Tell — each member demos their best OOP project
  • Group discussion: "What confused you most?"
  • Concept revision — quick 15-min recap
  • Announce Week 3 theme + project teams
Week 3 Data Tools & Environment Days 15–21
Day 15
Environment Setup
Build
  • Anaconda Navigator setup and walkthrough
  • Jupyter Notebook — cells, markdown, magic commands
  • VS Code Python extension setup
  • pip install: numpy, pandas, matplotlib, scikit-learn
  • Virtual environments explained
Day 16
NumPy — Arrays & Operations
Learn
  • Creating arrays — 1D, 2D, 3D
  • Array operations, broadcasting
  • Slicing and indexing in NumPy
  • Math functions: mean, std, sum, dot product
Day 17
Pandas — DataFrames
Learn
  • Series and DataFrame creation
  • Loading CSV files with pd.read_csv()
  • Selecting, filtering, sorting data
  • Explore a real dataset: Titanic or Iris
Day 18
Pandas — Data Cleaning
Learn
  • Handling missing values (dropna, fillna)
  • Renaming columns, changing dtypes
  • GroupBy and aggregation
  • Merging and joining DataFrames
  • Exercise: clean a messy dataset from scratch
Day 19
Regex & Text Processing
Learn
  • re module — match, search, findall
  • Patterns: digits, words, emails, URLs
  • Text cleaning pipeline for NLP
  • Exercise: extract emails and phone numbers from text
Day 20
Data Visualization
Build
  • Matplotlib basics — line, bar, scatter plots
  • Seaborn — heatmaps, pairplots, distributions
  • Visualise the Titanic or Iris dataset
  • 📊 Data Storytelling: make 3 charts that tell a story
Day 21
Kaggle Introduction + Dataset Sprint
Event
  • 🌐 Create Kaggle accounts as a group
  • Navigate Kaggle: datasets, kernels, competitions
  • Teams pick a dataset — 1-hour exploration sprint
  • Each team presents 3 insights from their data
Week 4 AI/ML Intro + Final Project Days 22–30
Day 22
Introduction to Machine Learning
Learn
  • What is ML? Supervised vs unsupervised
  • Training data, test data, validation
  • Overfitting vs underfitting — intuition
  • Scikit-learn overview and pipeline
Day 23
Linear & Logistic Regression
Learn
  • Linear regression — concept and math intuition
  • Logistic regression for classification
  • Build: predict house prices / pass-fail classifier
  • Evaluate: accuracy, MSE, confusion matrix
Day 24
Decision Trees & KNN
Learn
  • Decision tree — how splitting works
  • K-Nearest Neighbors — distance metrics
  • Compare both on Iris dataset
  • Tune hyperparameters: depth, K value
Day 25
Project Kickoff + Team Formation
Project
  • 🚀 Final project announced — teams of 2–3
  • Project options: spam detector, price predictor, sentiment analyzer
  • Teams submit project topic + dataset plan
  • Mentors assigned to each team
  • GitHub repo created per team
Day 26–27
Project Building Days
Project
  • Teams work on data collection and cleaning
  • Model training and evaluation
  • Daily 10-min standups: blockers + progress
  • Mentor drop-in hours (1 hr per day)
  • Push code to GitHub daily
Day 28
Project Polish + Presentation Prep
Project
  • Code cleanup and documentation
  • README writing on GitHub
  • Prepare demo slides (max 5 slides)
  • Workshop: "How to present technical work" (20 min)
Day 29
Dry Run + Feedback
Review
  • Each team presents in front of peers (5 min)
  • Structured peer feedback form filled
  • Mentors give final improvement tips
  • Last round of code fixes
Day 30
🎉 Final Showcase Day
Grand Event
  • 🏆 All teams present final projects (10 min each)
  • Judged on: clarity, code quality, creativity, results
  • Awards: Best Project, Best Code, Best Presentation
  • Club certificates distributed
  • Plan announced for Month 2 roadmap

Ongoing Club Activities

These run throughout the 30 days — they build community, consistency, and confidence alongside the technical content.

📣
Daily Code Drop
Every member shares one thing they wrote or learned today — a function, a bug fix, a visualization — in the group chat. No judgment, just progress.
Daily
🎤
Demo Friday
Every Friday, 2–3 members demo their work to the club for 5 minutes. Builds presentation skills and keeps everyone accountable.
Weekly — Fridays
🤝
Peer Pair Sessions
Randomly pair two members every week. They spend 30 min helping each other with doubts. Juniors + seniors mixed intentionally.
Weekly
🧠
Concept Corner
One member explains a concept to the group each week — in their own words, no slides required. Teaching is the best way to learn.
Weekly
⚡
Speed Challenge
Every Thursday: one problem, 20 minutes, fastest correct solution wins. Leaderboard maintained for the month. Prizes at end of month.
Weekly — Thursdays
🌐
Guest Talk Series
Invite alumni or working professionals for a 20-min talk on "My AI journey." Exposes students to career paths and real-world applications.
Bi-weekly
📚
Resource Drops
Curated cheat sheets, YouTube videos, and notebook templates shared every Monday. One link per topic — no overwhelming lists.
Weekly — Mondays
🏅
Streak Tracker
Members track their daily coding streaks on a shared board. 30-day streak = special recognition on showcase day. Keeps motivation high.
Daily tracking
🐙
GitHub Profile Sprint
Week 3 activity — every member sets up a GitHub profile, pins 3 repos, and writes a README. Builds a public portfolio from day one.
Week 3 — once
💬
Weekly Discussion Prompt
Every Sunday: one open question posted — "What AI application surprised you this week?" or "Where would you use ML in college life?" Gets students thinking beyond code.
Weekly — Sundays
🔍
AI News Roundup
One member volunteers each week to share 2–3 recent AI/ML news items with the group. Builds awareness of the field beyond academics.
Weekly
🎯
End-of-Month Showcase
Day 30 grand event — all project teams present, certificates given, awards announced. Invite faculty and seniors. Sets the tone for Month 2.
Day 30 — once
